HTML

  • HTML: Hypertext Markup Language
  • The head element contains information about the webpage that is not visible to the end user. Examples of elements within the head element are:
    • title
    • style
    • base
    • meta information
  • The body element represents the visible content shown to the user.

CSS

  • CSS: Cascading Style Sheets
  • A margin indicates the amount of space around the outside of an element.
  • Padding indicates the amount of space around the content inside the element.
  • You can use color names (semantic references) or hexadecimal codes (hex #), found here.
  • HTML and CSS files are linked using the link element in the HTML doc head.
  • Class attributes (like card) allow us to assign a rule using a class selector to a particular HTML element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code . launches the branch in VS Code for editing

JavaScript

  • JavaScript (JS): The brains of the operation.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Unlike CSS, JS is linked to the HTML between the closing footer and body elements in the HTML doc.
  • In order to execute a function, you must call it.
  • Since JS is executed in order, the calls should go at the bottom so you don't accidentally try to call a function before it is defined.
